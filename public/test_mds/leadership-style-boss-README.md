# Leadership Style Test - Technical Documentation

## 1. Test Overview

**Test Name:** Leadership Style Assessment ("What Type of Boss Are You?")  
**Test Code:** `leadership-style-boss`  
**Test ID:** `test_011`  
**Purpose:** Identify leadership personality type and management style  
**Measures:** 8 leadership types based on Big Five traits (Openness, Conscientiousness, Agreeableness)  
**Questions:** 36 questions (12 pairs + 24 statements)  
**Duration:** 6-8 minutes  
**Response Type:** Mixed (pairs, 5-point scale, frequency scale)  
**Weight in Analysis:** 10%

---

## 2. Test Description

As a leader, are you a no-nonsense go-getter or an empathetic shoulder to cry on? The way you lead has a lot to do with your personality type!

**What It Measures:**

This test identifies your unique leadership style based on the scientifically-validated Big Five personality model and leadership research. It measures three key personality dimensions that predict leadership style:

- **Openness** - Innovation vs. Tradition
- **Conscientiousness** - Organization vs. Flexibility  
- **Agreeableness** - Empathy vs. Decisiveness

**The Science of Leadership:**

Modern research shows that great leaders aren't born—they're developed. While certain Big Five traits correlate with becoming a leader (high Openness and Conscientiousness, low Neuroticism), what makes someone a *good* leader is their willingness to grow and develop skills over time.

**Essential Trait: Growth Mindset**

Effective leaders have a growth mindset (developed by Carol Dweck):
- Embrace challenges
- Accept feedback
- Persist through setbacks
- Commit to lifelong learning
- Understand and value all employees
- Learn from failures
- Encourage team growth

**Research-Backed:**

Studies show that leadership coaching can improve performance by 20-30%. This test helps you understand your natural leadership tendencies so you can develop into the leader you want to be.

## 3. Test Format

### Question Type 1: Pairs (12 questions)
Choose which phrase describes you better

**Format:**
```
[Option A] ○━━━━━● [Option B]
```

**Example:**
```
I like to discuss complex ideas with my team
○━━━━━●
I like to discuss details with my team
```

**Response:** Binary choice (A or B)

---

### Question Type 2: Frequency Scale (12 questions)
Indicate how often each statement is true

**Format:**
```
Statement: "I would rather try something new than do things the same old way"
Never [○] Rarely [○] Sometimes [○] Usually [○] Always [○]
```

**Scale:**
1. Never
2. Rarely
3. Sometimes
4. Usually
5. Always

---

### Question Type 3: Agreement Pairs (12 questions)
Choose which phrase describes you better (similar to Type 1)

**Format:**
```
[Option A] ○━━━━━● [Option B]
```

**Example:**
```
I trust tried and true methods
○━━━━━●
I question tried and true methods
```

## 4. The Eight Leadership Types

Each type is defined by scores on three Big Five dimensions:

### Type 1: The Influencer
**Big Five Profile:** High Openness, High Conscientiousness, High Agreeableness  
**Leadership Traits:** Empathetic, supportive, motivational, organized

**Description:**
Empathetic and supportive, the Influencer always sees the good in others. She's a superstar at coaching, developing, and identifying the potential in team members. Assuming best intent is her M.O., and her biggest aspiration is to inspire growth in others.

**Strengths:**
- Excellent at coaching and developing people
- Identifies future needs and potential
- Empathetic and supportive
- Organized with strong plans
- Self-development focused

**Challenges:**
- Struggles with making tough decisions
- May overlook practical details
- Can fail to think through resource constraints
- May not consider logistical limitations

**Level Up By:**
- Learning to say "no" and standing firm
- Building more detail into communication
- Adding specificity to action plans
- Balancing empathy with decisiveness

**Famous Example:** Oprah Winfrey  
**Quote:** "I am happiest when I feel that people are getting something [from me], learning something, enhancing themselves in a way they never thought of before."

## 7. Scoring System

### Dimension Calculation

```python
def calculate_dimensions(answers):
    """
    Calculate scores for O, C, A dimensions
    """
    scores = {
        "openness": 0,
        "conscientiousness": 0,
        "agreeableness": 0
    }
    
    # Question mapping
    question_map = {
        "openness": {
            "pairs": [1, 2, 3, 4, 25, 26, 27, 28],
            "frequency": [13, 14, 15, 16]
        },
        "conscientiousness": {
            "pairs": [5, 6, 7, 8, 29, 30, 31, 32],
            "frequency": [17, 18, 19, 20]
        },
        "agreeableness": {
            "pairs": [9, 10, 11, 12, 33, 34, 35, 36],
            "frequency": [21, 22, 23, 24]
        }
    }
    
    # Reverse scored frequency questions
    reverse_scored = [17, 18]
    
    for answer in answers:
        question_id = answer['question_id']
        question_type = answer['question_type']
        response = answer['response']
        
        # Find which dimension this question belongs to
        for dimension, q_ids in question_map.items():
            if question_type == 'pair':
                if question_id in q_ids['pairs']:
                    # Pair questions: A = -1, B = +1
                    if response == 'A':
                        scores[dimension] -= 1
                    else:  # response == 'B'
                        scores[dimension] += 1
                        
            elif question_type == 'frequency':
                if question_id in q_ids['frequency']:
                    # Frequency: 1-5 scale
                    value = response
                    if question_id in reverse_scored:
                        value = 6 - response
                    
                    # Convert to -2 to +2 scale
                    scores[dimension] += (value - 3)
    
    return scores
```

### Determine Leadership Type

```python
def determine_leadership_type(dimension_scores):
    """
    Determine one of 8 leadership types based on O, C, A scores
    """
    # Determine if each dimension is High or Low
    o = "high" if dimension_scores["openness"] > 0 else "low"
    c = "high" if dimension_scores["conscientiousness"] > 0 else "low"
    a = "high" if dimension_scores["agreeableness"] > 0 else "low"
    
    # Map to leadership type
    type_map = {
        ("high", "high", "high"): "influencer",
        ("high", "high", "low"): "visionary",
        ("high", "low", "low"): "innovator",
        ("high", "low", "high"): "muse",
        ("low", "low", "high"): "cheerleader",
        ("low", "low", "low"): "action_hero",
        ("low", "high", "high"): "giver",
        ("low", "high", "low"): "pillar_of_strength"
    }
    
    profile = (o, c, a)
    return type_map.get(profile, "influencer")
```

## 10. Database Schema

### leadership_questions

```sql
CREATE TABLE leadership_questions (
    id SERIAL PRIMARY KEY,
    question_number INTEGER NOT NULL,
    question_type VARCHAR(20) NOT NULL,  -- 'pair' or 'frequency'
    
    -- For pair questions
    option_a TEXT,
    option_b TEXT,
    
    -- For frequency questions
    statement TEXT,
    
    -- Scoring info
    dimension VARCHAR(20) NOT NULL,  -- 'openness', 'conscientiousness', 'agreeableness'
    reverse_scored BOOLEAN DEFAULT FALSE,
    
    created_at TIMESTAMP DEFAULT NOW()
);

-- Index for lookups
CREATE INDEX idx_leadership_questions_type ON leadership_questions(question_type);
CREATE INDEX idx_leadership_questions_dimension ON leadership_questions(dimension);
```

```sql
CREATE TABLE leadership_results (

CREATE INDEX idx_leadership_results_user ON leadership_results(user_id);
```
