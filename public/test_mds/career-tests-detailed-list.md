### 1. Holland Code (RIASEC) Assessment ⭐ ОБЯЗАТЕЛЬНЫЙ

**Категория:** Интересы и склонности  
**Статус:** Бесплатный  
**Длительность:** 15 минут  
**Вопросов:** 48  

**Описание:**
Тест основан на теории Джона Холланда о шести типах личности. Определяет профессиональные интересы и навыки, помогая найти подходящую карьеру.

**Что измеряет:**
- **R (Realistic)** - Реалистичный: работа с инструментами, механизмами, животными
- **I (Investigative)** - Исследовательский: анализ, решение проблем, научная работа
- **A (Artistic)** - Артистический: творчество, самовыражение, искусство
- **S (Social)** - Социальный: помощь людям, обучение, взаимодействие
- **E (Enterprising)** - Предприимчивый: лидерство, убеждение, управление
- **C (Conventional)** - Конвенциональный: организация, систематизация, точность

**Результат:**
Код из 2-3 букв (например, RIA, SEC), показывающий доминирующие типы личности

**Источник:**
https://www.openpsychometrics.org/tests/RIASEC/

**Для кого:**
- Школьники 9-11 классов
- Студенты, выбирающие специализацию
- Люди на карьерном перепутье
- Требуется средний уровень английского

**Интеграция в платформу:**
- **Тип:** iFrame embed или API
- **Формат результатов:** JSON с баллами по каждому типу (R, I, A, S, E, C)
- **Скоринг:** 0-100% по каждой категории
- **Вес в итоговом анализе:** 35%

**Примеры профессий по кодам:**
- **RIA:** Инженер, Архитектор, Программист
- **SEC:** Менеджер по продажам, HR, Предприниматель
- **AIS:** Исследователь, Аналитик, Дизайнер
- **SAE:** Учитель, Психолог, Социальный работник

**Связь с другими тестами:**
- Photo Career Quiz → прямая корреляция (Building = R, Thinking = I, etc.)
- Career Values Quiz → дополняет мотивационную сторону
- DiSC → показывает стиль работы в рамках выбранного типа

### 2. Photo Career Quiz ⭐ ОБЯЗАТЕЛЬНЫЙ

**Категория:** Визуальные предпочтения  
**Статус:** Бесплатный  
**Длительность:** 10 минут  
**Вопросов:** 30 пар изображений  

**Описание:**
Быстрый визуальный тест для тех, кто лучше воспринимает информацию через изображения. Отлично подходит для людей с ограниченным английским.

**Что измеряет:**
Те же 6 категорий что и Holland Code, но через визуальные предпочтения:
- **Building (R)** - Строительство, работа руками, технические задачи
- **Thinking (I)** - Исследования, анализ, научная работа
- **Creating (A)** - Творчество, дизайн, искусство
- **Helping (S)** - Помощь людям, медицина, социальная работа
- **Persuading (E)** - Убеждение, продажи, лидерство
- **Organizing (C)** - Организация, планирование, администрирование

**Результат:**
Топ-3 категории с процентным соотношением

**Источник:**
https://www.truity.com/test/photo-career-quiz

**Для кого:**
- Школьники с любым уровнем языка
- Визуалы
- Те, кто устал от текстовых тестов
- Люди с ограниченным английским

**Интеграция в платформу:**
- **Тип:** Custom implementation с нашими изображениями
- **Формат результатов:** JSON с баллами по 6 категориям
- **Скоринг:** Процентное распределение
- **Вес в итоговом анализе:** 20%

**Преимущества:**
✅ Быстрый и простой  
✅ Не требует языковых навыков  
✅ Интуитивно понятный  
✅ Хорошо работает для молодежи  

**Конверсия в Holland Code:**
```
Building → R (Realistic)
Thinking → I (Investigative)
Creating → A (Artistic)
Helping → S (Social)
Persuading → E (Enterprising)
Organizing → C (Conventional)
```

### 3. DiSC Assessment ⭐ ОБЯЗАТЕЛЬНЫЙ

**Категория:** Личность и стиль работы  
**Статус:** Бесплатный  
**Длительность:** 12 минут  
**Вопросов:** 24  

**Описание:**
Популярный тест, используемый работодателями для оценки стиля работы и коммуникации. Определяет, как человек взаимодействует с другими и справляется с задачами.

**Что измеряет:**
- **D (Dominance)** - Доминирование: напористость, результативность, прямолинейность
- **I (Influence)** - Влияние: энтузиазм, оптимизм, коммуникабельность
- **S (Steadiness)** - Стабильность: терпение, лояльность, командная работа
- **C (Conscientiousness)** - Добросовестность: точность, аналитика, системность

**Результат:**
Профиль из 1-2 доминирующих букв + баллы по каждой категории (0-100)

**Источник:**
https://www.truity.com/test/disc-personality

**Для кого:**
- Все возрастные группы
- Особенно полезен для понимания команд работы
- Студенты, готовящиеся к первой работе
- Люди, меняющие карьеру

**Интеграция в платформу:**
- **Тип:** API integration или custom quiz
- **Формат результатов:** JSON с баллами D, I, S, C
- **Скоринг:** 0-100 по каждой категории
- **Вес в итоговом анализе:** 25%

**12 расширенных профилей:**
1. **D** - Доминирующий (Challenger)
2. **Di** - Вдохновитель (Inspirational)
3. **DC** - Генератор результатов (Results-Oriented)
4. **I** - Влиятельный (Enthusiast)
5. **ID** - Мотиватор (Motivator)
6. **IS** - Советник (Counselor)
7. **S** - Стабильный (Supporter)
8. **SI** - Помощник (Helper)
9. **SC** - Планировщик (Planner)
10. **C** - Добросовестный (Analyst)
11. **CD** - Скептик (Skeptic)
12. **CS** - Стабилизатор (Stabilizer)

**Примеры профессий:**
- **D (Доминирование):** CEO, Предприниматель, Хирург, Юрист
- **I (Влияние):** Маркетолог, PR-менеджер, Продавец, Актер
- **S (Стабильность):** Учитель, Медсестра, HR, Социальный работник
- **C (Добросовестность):** Бухгалтер, Инженер, Аналитик, Программист

**Связь с Holland Code:**
- D + E (Enterprising) = Лидерские роли
- I + S (Social) = Работа с людьми
- S + S (Social) = Помогающие профессии
- C + I (Investigative) = Аналитические роли

### 4. Career Values Quiz 🔄 ОПЦИОНАЛЬНЫЙ

**Категория:** Ценности и мотивация  
**Статус:** Бесплатный  
**Длительность:** 8 минут  
**Вопросов:** 12 категорий  

**Описание:**
Определяет, что важно для человека в работе: деньги, признание, помощь другим, творчество и т.д. Помогает понять внутреннюю мотивацию.

**Что измеряет 12 ценностей:**
1. **Achievement** - Достижения и успех
2. **Independence** - Независимость и свобода
3. **Recognition** - Признание и статус
4. **Relationships** - Отношения и коллектив
5. **Support** - Поддержка и помощь другим
6. **Working Conditions** - Условия работы и комфорт
7. **Challenge** - Вызовы и сложные задачи
8. **Creativity** - Творчество и инновации
9. **Knowledge** - Знания и обучение
10. **Security** - Безопасность и стабильность
11. **Meaning** - Смысл и цель
12. **Collaboration** - Сотрудничество

**Результат:**
Топ-5 ценностей с рейтингом важности

**Источник:**
https://www.careerguidancecharts.com/career-values-quiz.html

**Для кого:**
- Все, кто хочет понять свою мотивацию
- Люди в выгорании (burnout)
- Те, кто чувствует несоответствие с работой
- Школьники для осознанного выбора

**Интеграция в платформу:**
- **Тип:** Custom quiz на нашей платформе
- **Формат результатов:** JSON с топ-5 ценностей
- **Скоринг:** Ранжирование 1-12
- **Вес в итоговом анализе:** 15%

**Конверсия в O*NET Work Values:**
```
Achievement → Challenge, Risk, Power
Independence → Freedom, Creativity
Recognition → Knowledge, Status
Relationships → Meaning, Collaboration
Working Conditions → Reward, Security
Support → Helping Others
```

**Примеры карьер по ценностям:**
- **Achievement + Challenge:** Предприниматель, Стартапер, Инвестор
- **Independence + Creativity:** Фрилансер, Художник, Писатель
- **Relationships + Support:** Психолог, Учитель, Коуч
- **Security + Working Conditions:** Госслужащий, Банковский сотрудник

**AI промпт:**

### 5. Big Five Personality Test (OCEAN) 🔄 ОПЦИОНАЛЬНЫЙ

**Статус:** Бесплатный  
**Длительность:** 15 минут  
**Вопросов:** 44  

**Описание:**
Научно обоснованный тест личности, измеряющий 5 ключевых черт характера. Один из самых надежных психометрических инструментов.

**Что измеряет (OCEAN):**
- **O (Openness)** - Открытость опыту: творчество, любопытство, новизна
- **C (Conscientiousness)** - Добросовестность: организованность, ответственность
- **E (Extraversion)** - Экстраверсия: общительность, энергичность
- **A (Agreeableness)** - Доброжелательность: сотрудничество, эмпатия
- **N (Neuroticism)** - Нейротизм: эмоциональная стабильность

**Результат:**
Баллы 0-100 по каждой из 5 черт + процентили

**Источник:**
https://www.truity.com/test/big-five-personality-test
https://openpsychometrics.org/tests/IPIP-BFFM/

**Для кого:**
- Студенты университетов
- Взрослые, ищущие глубокое понимание
- Те, кто интересуется психологией
- Карьерные консультанты

**Интеграция в платформу:**
- **Тип:** API или custom implementation
- **Формат результатов:** JSON с 5 scores (0-100)
- **Скоринг:** Процентили относительно популяции
- **Вес в итоговом анализе:** 15%

**Связь с карьерами:**
- **Высокая Openness:** Художник, Исследователь, Писатель, Ученый
- **Высокая Conscientiousness:** Бухгалтер, Менеджер проектов, Врач
- **Высокая Extraversion:** Продавец, Ведущий, PR-менеджер
- **Высокая Agreeableness:** Психолог, Учитель, Медсестра
- **Низкая Neuroticism:** Военный, Пилот, Хирург

**Научная база:**
✅ Подтверждено тысячами исследований  
✅ Используется в академической среде  
✅ Стабильность результатов во времени  
✅ Кросс-культурная валидность

### 6. Skills Self-Assessment 💰 ПЛАТНЫЙ (990 ₸)

**Категория:** Навыки (Hard + Soft)  
**Статус:** Платный  
**Длительность:** 20 минут  
**Вопросов:** 60  

**Описание:**
Комплексная оценка текущих навыков пользователя: от технических (программирование, дизайн) до мягких (коммуникация, лидерство). Показывает сильные стороны и пробелы.

**Что измеряет:**

**Hard Skills (30 вопросов):**
- Компьютерная грамотность
- Программирование
- Дизайн и визуализация
- Аналитика и данные
- Технические навыки
- Языки программирования
- Офисные программы
- Специализированное ПО

**Soft Skills (30 вопросов):**
- Коммуникация
- Лидерство
- Командная работа
- Тайм-менеджмент
- Решение проблем
- Критическое мышление
- Эмоциональный интеллект
- Адаптивность

**Результат:**
- Карта навыков (0-10 по каждому)
- Топ-10 сильных навыков
- Топ-10 навыков для развития
- Персонализированный план обучения

**Для кого:**
- Студенты, готовящиеся к работе
- Люди при смене карьеры
- Те, кто хочет составить резюме
- HR и карьерные консультанты

**Интеграция в платформу:**
- **Тип:** Собственная разработка
- **Формат результатов:** Детальный JSON с 60 навыками
- **Скоринг:** 0-10 по каждому навыку
- **Вес в итоговом анализе:** 10%

**Уникальность:**
✅ Сравнение с требованиями реальных вакансий  
✅ Gap analysis (что нужно для целевой профессии)  
✅ Интеграция с курсами (Coursera, Udemy, Stepik)  
✅ Трекинг прогресса со временем  

**AI-анализ навыков:**
```python
# Пример алгоритма
def analyze_skills(user_skills, target_career):
    required_skills = get_career_requirements(target_career)
    
    gaps = []
    strengths = []
    
    for skill in required_skills:
        user_level = user_skills.get(skill, 0)
        required_level = required_skills[skill]
        
        if user_level >= required_level:
            strengths.append(skill)
        else:
            gap = required_level - user_level
            gaps.append({
                'skill': skill,
                'gap': gap,
                'estimated_learning_time': calculate_time(gap)
            })
    
    return {
        'strengths': strengths,
        'gaps': gaps,
        'readiness_score': calculate_readiness(user_skills, required_skills)
    }
```

### 7. Emotional Intelligence (EQ) Test 💰 ПЛАТНЫЙ (990 ₸)

**Категория:** Эмоциональный интеллект  
**Статус:** Платный  
**Длительность:** 15 минут  
**Вопросов:** 40  

**Описание:**
Оценка эмоционального интеллекта - способности понимать свои и чужие эмоции, управлять ими. Критически важен для лидерства, командной работы и карьерного успеха.

**Что измеряет (модель Гоулмана):**
1. **Self-Awareness** - Самосознание: понимание своих эмоций
2. **Self-Management** - Самоуправление: контроль эмоций
3. **Social Awareness** - Социальная осведомленность: эмпатия
4. **Relationship Management** - Управление отношениями: влияние

**Результат:**
- EQ Score (0-200)
- Детализация по 4 компонентам
- Сравнение со средними показателями
- Рекомендации по развитию

**Источник:**
Custom implementation based on:
- Goleman's EQ framework
- Mayer-Salovey-Caruso EQ Test (MSCEIT)
- TalentSmart EQ Assessment

**Для кого:**
- Будущие менеджеры и лидеры
- Студенты бизнес-программ
- Те, кто работает с людьми
- Желающие улучшить отношения

**Интеграция в платформу:**
- **Тип:** Собственная разработка
- **Формат результатов:** JSON с 4 scores + общий EQ
- **Скоринг:** 0-200 (100 = средний)
- **Вес в итоговом анализе:** 10%

**Связь с карьерами:**
- **Высокий EQ:** Психолог, HR, Учитель, Менеджер, Лидер команды
- **Низкий EQ (это ОК):** Программист, Аналитик, Исследователь

**Примеры вопросов:**
```
1. Как часто вы замечаете, что чувствуете?
   - Всегда / Часто / Иногда / Редко / Никогда

2. Когда кто-то расстроен, я обычно:
   - Чувствую их эмоции
   - Пытаюсь помочь
   - Оставляю в покое
   - Не замечаю

3. В стрессовой ситуации я:
   - Сохраняю спокойствие
   - Немного волнуюсь, но контролирую
   - Сильно нервничаю
   - Теряю контроль
```

**Научная обоснованность:**
✅ Коррелирует с карьерным успехом (r=0.58)  
✅ Предсказывает эффективность лидерства  
✅ Можно развивать тренировками

### 8. Learning Style Inventory (VARK) 🆓 БЕСПЛАТНЫЙ

**Категория:** Стиль обучения  
**Статус:** Бесплатный  
**Длительность:** 10 минут  
**Вопросов:** 16  

**Описание:**
Определяет предпочитаемый стиль восприятия информации. Помогает выбрать наиболее подходящие методы обучения и профессии, где этот стиль будет преимуществом.

**Что измеряет (VARK модель):**
- **V (Visual)** - Визуальный: диаграммы, графики, видео
- **A (Auditory)** - Аудиальный: лекции, обсуждения, подкасты
- **R (Reading/Writing)** - Чтение/Письмо: тексты, заметки, статьи
- **K (Kinesthetic)** - Кинестетический: практика, эксперименты, движение

**Результат:**
Распределение предпочтений (может быть multimodal - несколько стилей)

https://vark-learn.com/the-vark-questionnaire/

**Для кого:**
- Школьники, выбирающие методы учебы
- Студенты, испытывающие трудности с обучением
- Те, кто планирует онлайн-курсы
- Все, кто хочет учиться эффективнее

**Интеграция в платформу:**

### 9. Work Motivation Scale 💰 ПЛАТНЫЙ (990 ₸)

**Категория:** Мотивация и драйвы  
**Статус:** Платный  
**Длительность:** 12 минут  
**Вопросов:** 36  

**Описание:**
Глубокий анализ внутренней и внешней мотивации. Что заставляет человека работать: деньги, признание, интерес, миссия? Помогает найти работу, которая будет мотивировать долгосрочно.

**Что измеряет:**

**Intrinsic Motivation (Внутренняя):**
- Интерес к задачам
- Любопытство и познание
- Мастерство и рост
- Автономия и свобода
- Цель и смысл

**Extrinsic Motivation (Внешняя):**
- Деньги и бонусы
- Признание и статус
- Продвижение по карьере
- Избегание наказаний
- Социальное одобрение

**Результат:**
- Баланс внутренней vs внешней мотивации
- Топ-3 драйвера
- Потенциальные демотиваторы
- Тип мотивационного профиля

**Основано на:**
- Self-Determination Theory (Deci & Ryan)
- Motivation and Personality (Maslow)
- Drive Theory (Daniel Pink)

**Для кого:**
- Люди в career burnout
- Те, кто чувствует отсутствие мотивации
- Студенты, выбирающие между деньгами и страстью
- Менеджеры, мотивирующие команды

**Интеграция в платформу:**
- **Тип:** Собственная разработка
- **Формат результатов:** JSON с intrinsic/extrinsic scores
- **Скоринг:** 0-100 по каждой категории
- **Вес в итоговом анализе:** 8%

**6 мотивационных профилей:**

1. **The Achiever** (Достигатор)
   - Драйв: Цели, результаты, прогресс
   - Профессии: Предприниматель, Спортсмен, Sales

2. **The Explorer** (Исследователь)
   - Драйв: Новизна, обучение, открытия
   - Профессии: Ученый, Журналист, Путешественник

3. **The Creator** (Создатель)
   - Драйв: Творчество, самовыражение
   - Профессии: Художник, Дизайнер, Писатель

4. **The Harmonizer** (Гармонизатор)
   - Драйв: Помощь, связи, баланс
   - Профессии: Психолог, HR, Медиатор

5. **The Provider** (Добытчик)
   - Драйв: Безопасность, стабильность, деньги
   - Профессии: Финансист, Врач, Инженер

6. **The Influencer** (Влиятель)
   - Драйв: Признание, статус, власть
   - Профессии: Политик, CEO, Лидер

```python
def generate_comprehensive_analysis(test_results):
    """
    Комбинированный анализ всех тестов
    """
    
    # 1. Holland Code → Основные интересы
    holland_code = test_results.get('holland', {}).get('code')  # RIA
    primary_interests = map_holland_to_interests(holland_code)
    
    # 2. Photo Quiz → Подтверждение интересов
    photo_results = test_results.get('photo', {})
    visual_preferences = get_top_categories(photo_results)
    
    # Проверка согласованности
    consistency_score = check_consistency(holland_code, visual_preferences)
    
    # 3. DiSC → Стиль работы
    disc_profile = test_results.get('disc', {}).get('dominant')  # D
    work_style = map_disc_to_style(disc_profile)
    
    # 4. Values → Мотивация
    top_values = test_results.get('values', {}).get('top_5', [])
    motivators = map_values_to_motivators(top_values)
    
    # 5. Big Five → Черты личности
    ocean_scores = test_results.get('big_five', {})
    personality_traits = analyze_ocean(ocean_scores)
    
    # 6. Skills → Текущие возможности
    skill_scores = test_results.get('skills', {})
    current_capabilities = assess_skills(skill_scores)
    
    # 7. EQ → Эмоциональные способности
    eq_score = test_results.get('eq', {}).get('total')
    emotional_strengths = analyze_eq(eq_score)
    
    # 8. Learning Style → Как учиться
    vark_style = test_results.get('vark', {})
    learning_approach = determine_learning_style(vark_style)
    
    # 9. Motivation → Что драйвит
    motivation_profile = test_results.get('motivation', {})
    drive_factors = analyze_motivation(motivation_profile)
    
    # 10. Strengths → Природные таланты
    top_strengths = test_results.get('strengths', {}).get('top_5', [])
    natural_talents = map_strengths(top_strengths)
    
    # СИНТЕЗ ВСЕХ ДАННЫХ
    synthesis = {
        'career_profile': {
            'primary_interests': primary_interests,
            'work_style': work_style,
            'motivators': motivators,
            'personality': personality_traits,
            'strengths': natural_talents
        },
        'top_careers': recommend_careers(
            holland_code, 
            disc_profile, 
            top_values,
            ocean_scores,
            skill_scores,
            top_strengths
        ),
        'development_plan': create_development_plan(
            current_capabilities,
            top_careers,
            learning_approach
        ),
        'insights': generate_insights(
            consistency_score,
            all_test_results
        )
    }
    
    return synthesis
```
